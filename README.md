# ZEN ATTAQ — showcase

Static one-page discography website for ZEN ATTAQ.

## Preview

Open `index.html` directly or run a local server from this folder:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Quantify v2

The supplied Quantify Bold v2.6 font is included in TTF format and connected
through the `@font-face` rule in `styles.css`.
